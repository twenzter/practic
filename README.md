git init - объявить гит репозиторий локальный



git status - проверить список изменённых файлов



git add ./ - сохранить все файлы с изменениями в репозиторий



git commit -m ... - сделать коммит всех сохранений



git log - посмотреть историю коммитов.



git remote https://... - добавить удаленный репозиторий к нам



git remote -v - проверить подключили ли мы удаленный репозиторий



git push -u origin master - объеденить ветки удаленного репозитория и локального



git push - выгрузить данные с локального на удаленный репозиторий



git clone https://... - скопировать чужой репозиторий (объеденять ветви через push -u не надо)
